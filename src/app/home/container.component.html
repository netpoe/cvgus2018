Blockchain Asset Registry

Create BAR Tags that represent any tangible asset in the blockchain.

Currently, this dApp focuses on displaying the Asset.sol smart contract information. This smart contract is an ERC proposal inspired from the popular ERC20 contract. The Asset contract implements ownership transfer cappabilities and can be extended to meet specific requirements, such as to be Locatable, Serialized or to have more complex information.

This dApp implements a decentralized schema. Which is a JSON document that gets stored in IPFS and can be read by the dApp.

blockchainassetregistry.com

SANDER1

As a musician and music composer, I decided to create my own ICO for my music project called Súper Ánder.

This dApp presents information of the SANDER1 smart contract. A custom ERC20 token were each unit represents 1 song, so users can invest in a fraction of a song. This is implemented in the SuperCrowdsale smart contract, which delivers SANDER1 tokens in exchange of a capped amount of ETH.
